# Employee Leave Management System

## Project Overview

The Employee Leave Management System is a web-based application that helps organizations manage employee leave requests digitally.

The system provides two roles:

- Admin
- Employee

Employees can apply for leave, view their leave history, edit or cancel pending requests, and track the status of their applications.

Admins can manage employees, manage leave types, review leave requests, and approve or reject leave applications.

---

## Project Objectives

- Eliminate manual leave management.
- Maintain leave records digitally.
- Provide role-based access.
- Simplify the leave approval process.
- Build a secure and scalable application.
